Header

The header is comprised of several components:

  • Top Level Navigation
  • Typeahead/Search
  • Shop All Flyout

Top Level Navigation

Home Depot logo takes up back to the home page, at the current size, there is room for 7 icons.

Typeahead/Search

Search field expands, keyboard comes up and overlay comes up over home screen. As soon as user begins typing, typeahead begins returning corresponding CATEGORIES, BRANDS, BUYING GUIDES, INSTALLATIONS, and PROJECT GUIDES

Shop All Flyout

Touch the shop all flyout button and the menu will drop down showing the top level categories.

Footer

The current footer consists of a linked button that will send the user to the full site.

Home Page

Current home page has a large scrollable content area to the left and a scrollable right rail area in landscape or scrollable footer area in portrait mode.
