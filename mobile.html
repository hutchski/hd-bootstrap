Anatomy

The mobile web site is made from stacked components. Each page gets the same header and footer. The mobile header includes: the HD logo, storefinder link/icon, account link/icon, and cart link/icon. Unlike the tablet experience which uses font glyphs, the mobile icons are located on the Mobile_Sprite.png.

Objects

Module/Pod

This module/pod is the building block of the site. The basic features of the module/pod include:

  • 1px solid #ccc border bottom
  • padding: 10px all around
  • white background

If you need a top border for the module add the class, border-top.

Variations include:

  • Background color
  • Border color
  • Font color

The secondary module is light gray. The secondary module is used in the buy-box and in cart and checkout.

The tertiary module is dark gray with white border and white text. Tertiary is used on the videos pages.

Primary list

This list is used throughout the site. You can see it on the home page in the footer, and on the category pages.

The basic features of the list include:

  • 1px solid #ccc border bottom
  • font-size: 18px
  • carrot

Variations include:

  • Background color of li
  • Border color of li
  • Font weight
  • Font Color

Nav list

This list is also used throughout the site. You can see it at the bottom of the footer and at the top of the category video pages.

The basic features of the list include:

  • display: inline block
  • font-size: 12px
  • border-right: 1px solid #000

Variations include:

  • carrot
  • Border color of li
  • Font weight
  • Font Color

Store Location and Account

PLP

The mobile PLP stacks products pods. Products are sortable by the sort by dropdown. The list can also be refined.

Tabs

Choose a tab to filter items shown to online or store only or all products.

Sort By Dropdown

Touch the Sort by dropdown and the tumbler appears.

Refine button

Touch the REFINE button and the refinements page is loaded like an overlay. The refinements are another variant of the primary list object.

PLP Pod

The PLP pod consists of an image, price, short description, ratings, ratings number.

The image is 100px x 100px

Storefinder

Upon visiting the site, user will be prompted to allow GPS. This will set the nearest local store automatically. If user refuses, the default store - 121 - will be set.

Landing Page

User gets to this page from the STORES link/icon in header.

Touch, USE GPS or enter Zip and user will be taken to page showing 20 nearest stores.

Touch Pin icon and larger map shows with pins. A list button shows to toggle between map/pin view and list view. Refine button opens the overlay.

Refine Overlay on Storefinder. User can choose with a drop down tumbler distance from current location or zip. also User can check services availalbe at store
